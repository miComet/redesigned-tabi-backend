# redesigned-tabi-backend

A backend service for redsigned-tabi-front

## To launch locally

### 1. install the dependencies

```bash=
$ pip -r install requirements.txt
```

### 2. launch service with uvicorn

```bash=
$ uvicorn main:app

```
